# Qlik Sense APIs (qliksense)
Collection of APIs for Qlik Sense, a business intelligence and visual analytics platform. Qlik provides REST APIs, WebSocket APIs, and developer tools for managing cloud tenants, applications, data connections, users, automations, and AI-powered analytics.

## APIs

### Qlik Engine API
JSON-RPC WebSocket API for the Qlik Associative Engine.

### Qlik Apps API
Manage Qlik Sense applications.

### Qlik Users API
Manage users, groups, and authentication.

### Qlik Spaces API
Manage shared and managed spaces.

### Qlik Data Connections API
Create and manage data connections.

### Qlik Automations API
Create no-code automation workflows.

### Qlik Reload API
Trigger and manage data reload operations.

### Qlik Webhooks API
Event-driven notifications from Qlik Cloud events.

### Qlik Reports API
Generate downloadable report assets.

### Qlik Machine Learning API
ML experiments, model deployment, and predictions.

### Qlik Natural Language API
Natural language query parsing.

### Qlik Tenants API
Tenant configuration and management.

### Qlik Audits API
Audit logging and compliance.
